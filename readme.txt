Plugin Name: Movie Reviews
Description: Movie Review post type with all the trimmings
Author: Justina Chan
Version: 1.0

Introduction
this is a WordPress plugin for user writes the movies review. In this plugin, user needs to put release year, director, IMDb link, photo and rating. it is can let the user manage the post with the content.

* register the custom post type 
* set configuration options for the custom post type 
*Create custom fields using core WordPress 
* interface with Meta Box for better custom fields 
* add a taxonomy for custom categories
